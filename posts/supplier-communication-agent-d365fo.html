Microsoft is gradually introducing agent-based experiences inside Dynamics 365 Supply Chain Management. One such capability is the Supplier Communication Agent, which enables intelligent, automated interactions with suppliers.

Prerequisites

Before setting up the Supplier Communication Agent, ensure the following:

  • D365 Supply Chain Management version 10.0.44 or later
  • Required features enabled in Feature Management
  • Latest Copilot components installed
Required Features:
  • Immersive Home
  • Agent Management
  • Supplier Communication Agent

Copilot Versions

  • Copilot for Finance and Operations Apps (v1.0.3048.2 or later)
  • Copilot in D365 Supply Chain Management (v1.1.03071.1 or later)

Set Up Agent Identity

It is recommended to create a dedicated agent identity instead of using a personal user account.

Required Licenses

  • Power Apps Premium
  • Power Automate Premium
  • D365 Supply Chain Management

Security Roles

Dataverse:

  • Finance and Operations Basic User
  • Supplier Communications Agent
  • Environment Maker

D365 Finance & Operations:

  • Supplier Communications Agent
  • System User

Why This Matters

By introducing supplier-facing agents, organizations can reduce manual follow-ups, improve communication consistency, and prepare their ERP systems for AI-driven workflows.

Final Thoughts

Supplier Communication Agent is an early but important step toward agentic ERP systems. As Microsoft expands Copilot and agent capabilities, this area will become critical for scalable supply chain operations.
